# environmentalsanitationsystem
北京环卫系统
=========

系统简介
-------
北京环卫系统主要是对北京环卫相关的一些设备和数据做统计，然后生成报表。主要包括垃圾粪便、应急管理、环卫车辆、厕所环卫设施、作业道路、其他台账
、人员收支、设施年报、综合报表
